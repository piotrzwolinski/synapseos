LAYER 2: PHYSICS & STRESSOR RULES — "How the World Works"
================================================================================

This is where the Graph becomes irreplaceable. These are PHYSICS RULES that
a pure LLM cannot reliably know or enforce. They are encoded as chains:

  Environment → EXPOSES_TO → Stressor → DEMANDS_TRAIT → Physical Trait
                                                ↑
                                    Material → PROVIDES_TRAIT

If the material does not provide the demanded trait → SYSTEM BLOCKS THE SALE.
No hallucination. No guessing. Deterministic enforcement.

─── ENVIRONMENT → STRESSOR CHAINS ───

  OUTDOOR/ROOFTOP
    ├── EXPOSES_TO → High Humidity
    │     └── (no specific trait demanded — general degradation risk)
    └── EXPOSES_TO → Outdoor Condensation
          └── DEMANDS_TRAIT → "Thermal Insulation"
          ⚡ Result: Only DOUBLE_WALL_INSULATED products (GDMI) pass.
                     GDP, GDB, GDC → BLOCKED for outdoor use.

  COMMERCIAL KITCHEN
    ├── EXPOSES_TO → Grease/Oil Exposure
    │     └── DEMANDS_TRAIT → "Mechanical Filtration"
    │     ⚡ Carbon alone (GDC) → BLOCKED. Needs pre-filter stage.
    └── EXPOSES_TO → Chemical Vapor Exposure
          └── DEMANDS_TRAIT → "Porous Adsorption"

  HOSPITAL / HEALTHCARE
    └── EXPOSES_TO → Hygiene Requirements
          └── DEMANDS_TRAIT → "Corrosion Resistance C5"
          ⚡ Förzinkat Z275 (C3) → BLOCKED.
             Must be Rostfritt, Syrafast, or ZM.

  PHARMACEUTICAL / CLEANROOM
    ├── EXPOSES_TO → Hygiene Requirements → DEMANDS → Corrosion Resistance C5
    ├── EXPOSES_TO → Chemical Vapor Exposure → DEMANDS → Porous Adsorption
    └── EXPOSES_TO → Particulate Exposure → DEMANDS → Mechanical Filtration

  WASTEWATER TREATMENT
    ├── EXPOSES_TO → Chemical Vapor Exposure → DEMANDS → Porous Adsorption
    └── EXPOSES_TO → H2S / Hydrogen Sulfide Corrosion
          ⚡ H2S destroys galvanized steel (FZ, C3). Requires C5 minimum.

  ATEX ZONE
    └── EXPOSES_TO → Explosive Atmosphere
          └── DEMANDS_TRAIT → "Electrostatic Grounding"
          ⚡ All metallic components must be grounded. Standard media BLOCKED.

─── MATERIAL CAPABILITY MATRIX ───

  Material              │ Corrosion Class │ Traits Provided
  ──────────────────────┼─────────────────┼────────────────────────────
  Förzinkat Z275        │ C3              │ Corrosion Resistance C3

LAYER 3B: INSTALLATION CONSTRAINTS — "Hard Engineering Limits"
================================================================================

These are constraints that BLOCK a configuration if physical limits are violated.
Encoded per product, with dynamic error messages.

  TYPE: SET_MEMBERSHIP (environment whitelist)
  ─────────────────────────────────────────────
  Products: GDP, GDB, GDC, GDC FLEX, GDMI
  Rule: Product can ONLY be used in whitelisted environments.
  Error: "Product not rated for {environment} installation.
          Uses {construction_type} construction which does not meet
          leakage class and hygiene requirements."
  Example: GDB (SINGLE_WALL) → blocked for Hospital.
           GDMI (DOUBLE_WALL_INSULATED) → allowed for Hospital.

  TYPE: CROSS_NODE_THRESHOLD (material vs. chlorine)
  ─────────────────────────────────────────────────────
  Products: GDP, GDB, GDC, GDC FLEX, GDMI
  Rule: Material's max chlorine tolerance vs. environment's chlorine level.
  Error: "Material tolerates max {threshold}ppm chlorine but environment
          has ~{input_value}ppm. Upgrade material."
  Example: FZ (C3) in swimming pool → chlorine 250ppm >> tolerance 50ppm
           → BLOCKED. System suggests Syrafast (C5.1).

  TYPE: CROSS_NODE_THRESHOLD (filter depth limit)
  ─────────────────────────────────────────────────
  Products: GDB, GDC
  Rule: Requested filter depth must fit in selected housing length variant.
  Error: "Specified depth {input_value}mm exceeds maximum {threshold}mm
          for the selected {housing_length}mm variant."
  Example: 600mm filter in GDB 550mm variant → BLOCKED.
           System suggests: "A longer variant accommodates this depth."

  TYPE: COMPUTED_FORMULA (service clearance)
  ──────────────────────────────────────────────
  Products: GDB, GDC, GDC FLEX
  Rule: Side-hinged door needs clearance = housing_width × service_factor.
  Error: "Service access requires {required}mm clearance but only
          {available}mm available."
  Severity: WARNING (not blocking, but flagged for installer)

  STEP 1: APPLICATION DETECTION
    Input: "swimming pool"
    Graph match: Application[Swimming Pool]
    Activates: Stressor[Chlorine Exposure], Stressor[High Humidity]
    Triggers: LogicGate[Chlorine Exposure Gate]

  STEP 2: COMPETITOR RESOLUTION
    Input: "CamCube CC"
    Graph match: CompetitorProduct[CamCube CC] → CROSS_REFERENCES →
                 ProductFamily[GDC Patronfilterskåp] (82% confidence)
    Result: detected_family = "GDC Patronfilterskåp"

  STEP 3: STRESSOR → TRAIT EVALUATION
    Chlorine Exposure → DEMANDS_TRAIT → Corrosion Resistance C5
    GDC default material = Förzinkat Z275 (C3)
    C3 does NOT provide C5 → CONSTRAINT VIOLATION

  STEP 4: INSTALLATION CONSTRAINT CHECK
    IC_MATERIAL_CHLORINE fires:
    "Material tolerates max 50ppm chlorine but swimming pool has ~250ppm.
     Upgrade material."
    IC_ENVIRONMENT_WHITELIST: GDC allowed for Indoor, ATEX → Swimming Pool
    is not in whitelist → SECOND VIOLATION

  STEP 5: CAPACITY SIZING
    5,000 m³/h needed. GDC capacity table:
    1500x600 → 5,000 m³/h (40 cartridges) ← exact match
    Result: 1 module, size 1500x600

  STEP 6: ALTERNATIVE GENERATION (Sales Recovery)
    Since GDC is blocked by material constraint:
    → Alternative 1: GDC in Zinkmagnesium ZM310 (C5) — same product, upgraded
    → Alternative 2: GDC in Rostfritt 1.4301 (C5) — same product, upgraded
    → Alternative 3: GDC in Syrafast 1.4404 (C5.1) — premium, chlorine-proof

  STEP 7: VERDICT
    "GDC Patronfilterskåp in standard material (FZ) is NOT suitable for
     swimming pool environment due to chlorine corrosion. Recommended:
     upgrade to Zinkmagnesium ZM310 (C5) or Syrafast 1.4404 (C5.1)
     for chlorine resistance. Configuration: 1× 1500x600mm module,
     40 cartridges, 5,000 m³/h capacity."

  ⚡ ALL of the above is graph-driven. The LLM only formats the response.
     It CANNOT hallucinate the chlorine threshold, the material properties,
     or the cartridge count. Those are FACTS from the Graph.
